To run the code locally, following files will be used. All files should be in the same path to run the website.

* app.py - The main backend code file.

* lang_model.pkl - Language model file which is loaded in the app.py. This file must be in the same path as app.py

* logistic_model.pkl - Logistic Regression model file which is loaded in app.py. This file must be in the same path as app.py

* templates folder - contains all the html files needed to run the website.

* static folder - contains all the css, javascript, images files which are needed to run the website.

* Milestone2.mp4 - This is the video file in which I have demonstrated the website that is created on a high level.
